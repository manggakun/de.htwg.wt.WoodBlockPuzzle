About WoodBlockPuzzle

Origin

WoodBlockPuzzle is a game based on the idea of Tetris.
Blocks must be placed in a limited space and, if possible, form closed rows which then dissolve. Full rows vanish an the player get points for that. This goes on until the new blocks appear too fast to be used properly. The player gets under time pressure, makes mistakes and finally there is no more room for new blocks.
Game over.

Difference to Tetris

WoodBlockPuzzle can be viewed as a game with a tetris-like feel, without the aspect of time pressure.
Instead of moving blocks and the idea to insert these blocks quickly, everything is built like a puzzle, which can be thought about as long as you want.
The size of the gird is 8x8 in this case.

Collect Points

Full rows or columns earn points for the player. After that the cells are free again.
If all three existing blocks have been used, three new ones will appear and so on.

End of the Game

The game could theoretically continue endless, but it is over when the player can no longer use any of his given blocks. He is forced to give up. This saves the player's current high score, which he then has to beat in a new attempt.
